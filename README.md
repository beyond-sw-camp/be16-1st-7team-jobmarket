## 📌 프로젝트 개요

## 프로젝트명: 채용 플랫폼 기반 데이터베이스 설계 (사람인 클론 DB)

설명: 이 프로젝트는 채용 사이트(예: 사람인)를 모델로 한 데이터베이스를 설계 및 구현하는 것을 목표로 합니다. 사용자, 기업, 채용공고, 지원 이력 등 주요 엔티티를 포함한 ERD 모델링과 DB 아키텍처를 수립하고, 이를 기반으로 테스트용 DML 쿼리를 작성합니다.
